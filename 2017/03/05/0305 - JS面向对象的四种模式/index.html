JS面向对象的四种模式

四种模式

工厂模式

  1. 声明一个对象
  2. 向对象添加属性和方法
  3. 返回这个对象
  4. 创建新对象，通过函数创建
  1. 在一个函数内创建一个对象，给对象添加属性和方法，再将对象返回
  2. 无法识别对象的类型，全部为Object

构造函数模式

  1. 首字母大写
  2. 没有显式创建对象
  3. 直接赋值属性方法给this
  4. 没有return语句
  5. 使用new操作符创建对象
  6. instanceOf()能够识别对象

原型模式

  1. 省略了参数，所有实例均取得了相同的属性
  2. 一个实例修改了引用，另一个实例也随之更改了引用

原型模式+构造函数模式

  1. 构造函数定义实例属性
  2. 原型模式用于定义方法和共享属性
  3. 每个实例都有自己的一份实例属性，并共享着方法
