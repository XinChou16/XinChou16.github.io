CSS - 三列布局

圣杯布局

布局一览,实现的是三列布局,左右定宽,中间设置为100%,自适应,基本思想是通过把中间放在最上面进行渲染,然后给左右设置margin, position 来不压住中间自适应的内容.

要点总结

  • HMTL中,中间盒子最先渲染
  • 三个盒子均左浮动
  • 左右两个盒子均设置负的自身左偏移
  • 左右两个盒子均设置成相对定位,距离相应的边界自身的宽度值
  • 左右两个均设置为负值是因为左右会压住中间盒子的内容,圣杯通过定位的方式来实现的,圣杯与双飞翼的不同就在于此,
  • 圣杯布局和双飞翼布局解决问题的方案在前一半是相同的，也就是三栏全部float浮动，但左右两栏加上负margin让其跟中间栏div并排，以形成三栏布局。
