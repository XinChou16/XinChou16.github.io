2015 Baidu IFE Task 0002

终于用上了markdown,COOL

任务说明

实现一个小需求,在输入框输入数字,数字没有进行数字判断,值做简单的加法运算,然后网页显示结果.JS应该是这样的,

功能说明

1.首先,页面onload函数不写,根本不会有反应,逻辑应该是这样的,当页面完全load,执行代码内容,然后调用函数initEvent(),执行加法功能;
2.add()函数,简单的加法函数,但实际测试发现是字符串拼接,这个疑问,有待解答.
3.renderResult(),渲染结果输出到网页上,主体就是获取ID的内联元素,然后使用innerHTML方法,实现网页显示的功能.
4.addEventHandle().获取输入框的值,没有进行强制数字转换,造成,输入数字时,只会进行字符串拼接,应该是这个原因吧,首先,声明了三个值,然后调用renderResult()方法;
5.添加事件绑定函数,给按钮绑定事件,接收3个参数,事件名称,要执行的函数,flase表示在冒泡阶段调用事件的处理函数.
6.最后,全局调用,实现功能.

最后,小结一下,还是踩在了onload坑上,还有addEventHandle()的主体功能上.
